## 2. 本次精读原文

**官方短摘录：**

> Codex CLI is OpenAI's coding agent that you can run locally from your terminal.

**本次英文精读材料（基于官方页面信息改写）：**

Codex CLI is a local coding agent from OpenAI. You use it in a terminal, where it can inspect the selected project folder, suggest changes, edit files, and run commands. The tool is designed for developers who want an AI teammate close to their code, not only in a web app. To get started, you install the CLI with a package manager such as npm, run the codex command, and sign in with a ChatGPT account or an API key. New versions are released regularly, so the documentation also explains how to upgrade and where to read release notes.

## 3. 自然中文翻译

Codex CLI 是 OpenAI 的一个本地编码代理。你在终端里使用它，它可以查看你选中的项目文件夹，提出修改建议，编辑文件，并运行命令。这个工具是给开发者用的：当你希望 AI 搭档更靠近你的代码，而不只是停留在网页应用里，就可以使用它。开始使用时，你需要用 npm 这样的包管理器安装 CLI，运行 `codex` 命令，然后用 ChatGPT 账号或 API key 登录。Codex CLI 会定期发布新版本，所以官方文档也会说明如何升级，以及在哪里查看版本说明。

## 7. 高频句式

### 句式 1：X is a Y from Z

意思：X 是来自 Z 的一种 Y。

例句：

- Codex CLI is a local coding agent from OpenAI.
  → Codex CLI 是 OpenAI 的一个本地编码代理。
- Supabase is a backend platform for developers.
  → Supabase 是一个面向开发者的后端平台。
- Claude Code is a coding tool from Anthropic.
  → Claude Code 是 Anthropic 的一个编码工具。

还能这样用：

官方文档开头常用这个句式给产品下定义。读的时候先抓 `X is Y`，再看后面的来源或修饰。

### 句式 2：You use it in X, where it can...

意思：你在 X 这个环境里使用它，在那里它可以做某些事。

例句：

- You use it in a terminal, where it can run commands.
  → 你在终端里使用它，在那里它可以运行命令。
- You use it in a browser, where it can open web pages.
  → 你在浏览器里使用它，在那里它可以打开网页。
- You use it in an editor, where it can read your code.
  → 你在编辑器里使用它，在那里它可以读取你的代码。

还能这样用：

`where` 不一定只翻译成”哪里”，它经常把一个场景和这个场景里的能力连接起来。

### 句式 3：To get started, you...

意思：开始使用时，你需要……

例句：

- To get started, install the package.
  → 要开始使用，先安装这个包。
- To get started, create an account.
  → 要开始使用，先创建一个账号。
- To get started, run the setup command.
  → 要开始使用，先运行设置命令。

还能这样用：

看到 `To get started`，后面通常是入门步骤，不是抽象概念。

### 句式 4：X are released regularly, so...

意思：X 会定期发布，所以……

例句：

- New versions are released regularly, so read the changelog.
  → 新版本会定期发布，所以记得看更新日志。
- Updates are released regularly, so keep your tool current.
  → 更新会定期发布，所以让你的工具保持最新版。

还能这样用：

`so` 后面通常接结果或建议。读技术文档时，先判断前后是不是因果关系。
